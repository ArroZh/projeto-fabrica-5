## 🚀 Sobre

Projeto desenvolvido em Python para reproduzir uma playlist de músicas em sequência. O foco foi praticar manipulação de arquivos, organização do código e reprodução de áudio.

## ✨ Funcionalidades

- Reprodução automática
- Execução sequencial
- Leitura de arquivos de áudio
- Interface simples
- Código modular
